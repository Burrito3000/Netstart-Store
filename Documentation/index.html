OnScroll Animation

You need to add a class called ".animate" on the element you want to animate.

Then you need to add some date attribute to configer your animation

To select animate from the direction you want just use (data-left,data-right,data-top,data-bottom) like this (data-left="100") the value will be caculeted in pixels

You can also define animation duration, delay, type/timing(css3 animation like linear,ease,ease-in,ease-out,ease-in-out,step-start,step-end)
(data-duration="1" data-delay=".7" data-animate-type="linear")
